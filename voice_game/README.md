voice game
==========
使用verilog编写的基于NEXYS4DDR的FPGA的小游戏
------
#### keyword: VGA, Microblaze, ROM, PWM
<br>
游戏的目的是通过声音来控制物体的跳跃，来避开到来的坑<br>
当物体掉进坑时，游戏结束<br>
<br>
<br>
code中有三个文件夹，分别放的是SDK的c文件，verilog和VHDL代码文件，以及bit文件
<br>
